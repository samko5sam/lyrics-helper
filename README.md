# Lyrics Helper

有趣的學習小工具，使用者可以搜尋歌詞，在歌詞頁面選取任一單字會出現單字解釋，選取一段文字會翻譯成中文。遇到單字使用者可以將單字加入單字本，以便再次複習。

- 搜尋歌詞
- 選取進行單字解釋
- 選取文字進行翻譯
- 儲存單字到單字本
